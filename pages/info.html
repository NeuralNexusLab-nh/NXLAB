CORE COMMANDS MANUAL

Complete reference manual for NXLAB Console v1.0.0 hosted on nxlab.nett.to

// NETWORK UTILITIES

ip
Retrieve the client's public IP address as recorded by the server node.
Usage: ip
http
Sends an HTTP request to the designated host. Defaults to secure HTTPS. Automatically falls back to standard HTTP if secure handshake fails.
Usage: http [url_or_domain]
tcp
Attempts to open a direct raw TCP socket stream to a specified host and port to check port availability. Timeout at 3000ms.
Usage: tcp [host] [port]
udp
Dispatches an arbitrary UDP datagram containing a custom text payload to a targeted remote port.
Usage: udp [host] [port] [message_payload]
nslookup
Performs high-speed DNS hostname resolution utilizing Node's native lookup bindings. No sub-process execution.
Usage: nslookup [domain]

// CRYPTO & ENCODING

base64ec
Converts raw text into a Base64-encoded string representation.
Usage: base64ec [raw_text_payload]
base64dc
Decodes a valid Base64 string back into standard human-readable format. Output is sanitarily escaped.
Usage: base64dc [base64_encoded_payload]
md5
Generates a secure 128-bit cryptographic MD5 hash signature of the input text payload.
Usage: md5 [text_to_hash]

// MATHEMATICAL ENGINE

sum
Performs floating-point addition of two input arguments.
Usage: sum [num1] [num2]
subtract
Performs floating-point subtraction of the second argument from the first.
Usage: subtract [num1] [num2]
multiply
Performs floating-point multiplication of two input arguments.
Usage: multiply [num1] [num2]
divide
Performs floating-point division. Division by zero is rejected.
Usage: divide [num1] [num2]
power
Raises a base floating-point number to the power of the specified exponent.
Usage: power [base] [exponent]

// UTILITIES

echo
Echoes back the entered text payload. Output is safely sanitized.
Usage: echo [message]
textarea
Formats multi-line or single-line text output neatly inside a retro terminal ASCII mockup box.
Usage: textarea [your_custom_text]
